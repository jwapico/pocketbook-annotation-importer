# Pocketbook Annotation Importer for Obsidian

Scans the notes directory on your Pocketbook and imports the highlights and annotations from each file into a new obsidian note.

Make sure to set the full path to your Pocketbook's notes directory.

You can run the sync from the button in settings or with the `Pocketbook Annotation Importer: Sync Pocketbook Notes` command.

## Manually installing the plugin

- Copy over `main.js`, `styles.css`, `manifest.json` to your vault `VaultFolder/.obsidian/plugins/your-plugin-id/`.
